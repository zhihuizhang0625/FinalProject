# WordleGame-Clone
[CIT594_Final Project.pdf](https://github.com/zhihuizhang0625/WordleGame-Clone/files/12680339/CIT594_Final.Project.pdf)

Our project is inspired by Wordle game created by New York Times and we would like to build a helper tool for recommending words. For this project, we created a GUI panel that simulates the Wordle website and a recommendation system to offer advice to users.
